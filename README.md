## husky_malloc
Free-list based memory allocator written in C. Part of coursework of CS3650 (Computer Systems)

### to compile - 
1. ```javascript git clone https://github.com/ritvikverma/husky_malloc.git ```
2. ```bash cd husky_malloc ```
3. ```bash make all ```

### to test - 
1. steps 1-3
2. ```bash perl test.pl```
